# Token Forest — 常见问题 ❓

[English](FAQ.md) · [简体中文](FAQ.zh-CN.md)

## 基础

### 一句话说，Token Forest 是什么？
一个像素桌宠：一棵住在任务栏上方的树，靠你在 Claude Code / Codex 里花掉的 token 长大。

### 免费吗？
免费——Token Forest 在公开测试期间免费，核心体验也会一直免费。

### 支持哪些平台？
Windows 10/11 和 macOS。Linux 在我们的关注列表里，但暂未正式支持。

### 它能追踪哪些 AI 工具？
目前是 Claude Code 和 Codex。更多助手正在考虑中——[来告诉我们你在用哪个](../../../issues/new/choose)。

## 隐私

### 它会读我的代码吗？
**不会。** 它只读这些工具本来就存在本机的用量日志里的 token *数量*——永远不碰文件内容、提示词或对话。详见[隐私说明](PRIVACY.zh-CN.md)。

### 需要联网吗？
不需要。所有功能完全离线可用。唯一的联网功能是排行榜，默认关闭，除非你主动开启——退出时还会删除你的记录。

### 源代码开放吗？
应用在私有仓库开发，目前不开源。我们在考虑开放核心组件（比如用量读取器），让隐私承诺可以被独立验证。

## 树

### 树长得有多快？
它随你的用量走过 8 个阶段。高强度的 agent 会话会让它肉眼可见地往上蹿；清闲的一天叶子几乎不动。具体阈值是游戏乐趣的一部分——都说破了还有什么意思？

### 为什么我的 token 数字这么大？
Token Forest 统计**全部四类 token**——输入、输出、缓存读、缓存写——和各类用量面板口径一致。Agent 类工具每次工具调用都要重读会话上下文，所以缓存读占比极高，总数轻松涨到十亿级。这是正常的。面板的花费视图会按每类的真实单价加权，所以*钱*的数字是诚实的。

### 应用关着的时候 token 算吗？
目前树只在应用运行时生长。"离线补长"在[路线图](ROADMAP.zh-CN.md)上。

### 我能同时有好几棵树吗？
你可以解锁并在**四个**树种之间切换（苹果、樱花、仙人掌，以及 🎄 圣诞树——每种各自保留自己的进度、果实和装饰）。收获当前树的果实就会解锁下一个树种，圣诞树位于进阶链的最末端。而第五个树种，已经在温室里悄悄发芽。

### 果实有什么用？
成熟的树会随时间产出果实。点击收获，再用果实在商店里买装饰装扮你的树。

### 树挡住我的窗口了！
切到**胶囊模式**——角落里的一枚小状态药丸——或者关掉置顶，或者干脆把树拖到别处。你的森林永远不该和你的工作抢地方。

## 排行榜

### 排行榜会上传什么？
昵称、地区、当前树种、树的 token 总量和成长阶段——仅此而已，且绑定在一个匿名账户上。详见[隐私说明](PRIVACY.zh-CN.md)。

### 怎么退出排行榜？
在应用里关掉它。你的记录会从服务器**删除**，而不只是隐藏。
